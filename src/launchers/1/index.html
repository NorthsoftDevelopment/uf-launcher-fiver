Descripcion

Gamership Network es un proyecto de ecosistemas de torneos de Gamership dentro de un servidor de Minecraft, aplicando todas las funcionalidades, incluyendo membresias al ecosistema principal la Gamership Network, este ecosistema cuenta con varios tipos de modalidades las cuales amplian el contenido principal y la innovacion respecto a el contenido que pueda ofrecer
